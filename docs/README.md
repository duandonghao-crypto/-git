# 电费电量管理平台 — 知识库

## 系统概述

电费电量管理平台是一个基于 Flask + PostgreSQL 的多人协作电费管理系统。支持邮件下载电费账单、PDF 数据提取、企业分类、应收账款管理。

### 技术栈

| 层级 | 技术 |
|------|------|
| 后端 | Flask (Python 3) |
| 数据库 | PostgreSQL (Neon 云数据库) |
| 前端 | 原生 HTML/JS + 看板仪表盘 |
| 部署 | Render.com (免费) |

### 访问地址

- 公网：`https://electricity-manager-jxmc.onrender.com`
- 局域网：`http://192.168.240.83:8018`（本机部署时）

## 用户操作指南

### 登录

1. 打开看板首页
2. 顶部黑色栏点击「切换用户」
3. 选择已有用户名，或输入新名字创建
4. 无需密码，选名字即登录

### 电费支出流程（1→9）

| 步骤 | 按钮 | 说明 |
|------|------|------|
| 1 | 扫描邮箱 | 输入邮箱信息，统计电费账单和发票数量 |
| 2 | 下载电费单 | 从邮箱下载电费单 PDF 到附件目录 |
| 3 | 获取发票链接 | 收集发票 PDF 下载链接，手动下载发票 |
| 4 | 上传发票 | 选择下载好的发票 PDF 文件，自动重命名 |
| 5 | 去重 | 按 MD5 删除重复 PDF |
| 6 | 添加简称 | 从映射表（简称对应表.xlsx）读 B 列简称，追加到文件名 |
| 7 | 提取数据 | 从 PDF 第一页提取本期电费和电量，生成汇总 Excel |
| 8 | 分类文件 | 按公司名将文件移到对应子文件夹 |
| 9 | 上传到库 | 将汇总 Excel 数据写入数据库 |

### 收费录入

1. 选所属年月
2. 填入企业名称、户号、电费金额、电量
3. 点「上传」写入数据库

### 映射文件

`简称对应表.xlsx` 结构：
- A 列：户号（如 `[1100002327018]`）
- B 列：简称（如 `众达特`）
- C 列：公司全称（如 `北京市长城伟业投资开发有限公司`）

## API 参考

### 认证

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/auth/login` | 登录，body: `{"name": "管理员"}` |
| GET | `/api/auth/me` | 获取当前用户 |
| GET | `/api/auth/users` | 用户列表 |

### 核心数据

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/stats` | 统计数据（活跃户号、收支总额、应收） |
| GET | `/api/meters` | 户号列表 |
| GET | `/api/customers` | 企业列表 |
| GET | `/api/receivables` | 应收账款 |
| GET | `/api/search?q=xxx` | 流水查询 |

### 电费支出

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/expense_scan` | 扫描邮箱 |
| POST | `/api/expense_download_bills` | 下载电费单 |
| POST | `/api/expense_extract` | 提取 PDF 数据 |
| POST | `/api/expense_upload` | 上传提取数据到库 |
| POST | `/api/expense_upload_file` | 本地上传 Excel |

## 故障排除

| 问题 | 原因 | 解决 |
|------|------|------|
| 网页打不开 | 免费版休眠 | 等 10-20 秒刷新 |
| 提取数据卡住 | PDF 损坏 | 刷新重试，系统自动跳过坏文件 |
| 上传 0 条 | PDF 无法提取文字 | 确认 PDF 是国网原始格式 |
| 邮箱登录失败 | 密码错误 | 检查邮箱配置，端口 993 |
| GitHub 推送失败 | 国内网络 | 开 VPN 或多试几次 |
